# Calculadora de Gorjeta

## Sobre
App que calcula o valor da gorjeta com base no valor da conta e com quantos % desse valor o usuário deseja pagar a mais.

Aprendi e apliquei conceitos de:
* `Constraint Layout`
* `ImageView`
* `TextView`
* `TextInputEditText`
* `SeekBar`
* `Toast`
